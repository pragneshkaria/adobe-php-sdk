Libraries

There are 4 libraries that you need to run the samples.
  1. AMFPHP - This SDK requires version 1.9 beta or higher. Download this and install it in the amfphp/ directory. It is included with the .zip file, but not included with the code in the SVN repository.
  2. Flex / Ajax Bridge - This is installed for you.
  3. Adobe Spry Framework - Download this and install to the spry/ directory. It is included with the .zip file, but not included with the code in the SVN repository.
  4. WebORB for PHP from the Midnight Coders. Download this and install to the weborb/ directory. It is included with the .zip file, but not included with the code in the SVN repository.
